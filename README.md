# Mohamed Gamal Portfolio

A personal portfolio and blog built with Next.js.

## Overview

This project is intended to showcase:

- featured projects
- professional experience
- blog content
- contact and profile information

The app uses Next.js App Router, TypeScript, Tailwind CSS v4, and shadcn/ui.

## Development

```bash
pnpm install
pnpm run dev
```

## Quality

```bash
pnpm run lint
pnpm run fmt:check
pnpm run typecheck
```
