# Base UI React Skill

A compact skill package for implementing Base UI React components from `@base-ui/react`.

## Contents

- `SKILL.md` routes agents to the correct implementation patterns.
- `references/` contains grouped implementation notes.
- `examples/` contains copy-adaptable React examples.

## Intended usage

Use this skill when building custom-styled accessible React UI using Base UI primitives: dialogs, menus, forms, selects, comboboxes, tabs, toasts, RTL/CSP providers, and composition patterns.

## Key reminder

Base UI is unstyled. Agents must implement visual styling, focus states, responsive behavior, and project-specific design tokens.
