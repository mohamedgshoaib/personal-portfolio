## Problem

Study notes tend to scatter across documents, apps, and browser tabs. Rootly was built to keep learning material, progress, and review in one place without making the system feel like another course to manage.

## Solution

I built a notes and learning tracker with offline and cloud modes, progress dashboards, quiz-style review, and JSON/CSV exports.

## Architecture

The app uses Next.js 15 App Router, React 19, TypeScript, and Tailwind CSS 4. Supabase handles PostgreSQL, Auth, and RLS, Recharts powers the dashboards, and Radix UI with shadcn/ui covers the accessible component layer.
