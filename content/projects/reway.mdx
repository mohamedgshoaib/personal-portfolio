## Problem

Browser bookmarks are easy to save and hard to use later. Reway was built around a simpler idea: capture links quickly, add useful metadata, organize them in one place, and avoid being trapped in one browser's bookmark system.

## Solution

I built a bookmark manager with paste-to-bookmark extraction, grouped collections, real-time Supabase sync, and a Chrome MV3 extension for quick page capture and bulk actions.

## Architecture

The app uses Next.js 16 App Router, React 19, Tailwind CSS, and shadcn/ui. Supabase handles PostgreSQL, Auth, and Realtime; the extension handles browser capture; optimistic UI keeps the core interactions feeling immediate.
