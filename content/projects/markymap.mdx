## Problem

Most visual note tools make you manage the tool before you can think. I wanted Markymap to stay closer to writing: type Markdown, see the structure, and keep moving without an account or a heavy setup step.

## Solution

I built a split-screen editor that turns Markdown into an interactive mindmap as you type. It includes local auto-save, import/export, theme toggling, keyboard shortcuts, and expandable branches for larger maps.

## Architecture

The app uses Next.js 16 App Router, React 19, TypeScript, and Tailwind CSS 4. Coss UI and Base UI handle the interface primitives, Markmap renders the SVG map, the Web Audio API adds small interaction sounds, and Oxlint/Oxfmt keep feedback fast while working.
