## Problem

Mo's Experiences needed more than a brochure site. The team needed multilingual travel content, searchable destinations and packages, inquiry flows, and a way to manage the site without editing code.

## Solution

I delivered an English and French site with browsing, search, favorites, several inquiry forms, and an admin dashboard for managing content and watching analytics, Web Vitals, and SEO signals.

## Architecture

The project uses Next.js 15.5.5 App Router, TypeScript, Tailwind CSS 4, and shadcn/ui. Supabase handles PostgreSQL and Auth, TanStack Query manages server state, Resend sends email, `next-themes` handles theming, and a small custom layer tracks analytics.
